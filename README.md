Simple web browser, using QtWebEnginge and PyQt5. Forked from n303p4/muribonbon .

Features:
  
  * Browser  
  * QtWebEngine as web enigne  
  * History logging  
  * Integrated AdBlocker  
  * Fast, but battery-saving  
  * Downloads (finally)  
  * Private mode
